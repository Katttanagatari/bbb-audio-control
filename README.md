# bbb-audio-control
extension for audio volume control for SSAU BBB subdomains

Расширение для управления громкостью в BigBlueButton конференциях СГАУ (SSAU). Устанавливает единую громкость для аудио и видео.

## Установка
Скачайте расширение через git clone или code -> download zip
```md
git clone https://github.com/katttanagatari/bbb-audio-control
```
Для Chrome
```md
chrome://extensions/
```
Включить режим разработчика -> Загрузить распакованное

Edge
```md
edge://extensions/
```
Включить режим разработчика -> Загрузить распакованное

Firefox
```md
about:debugging#/runtime/this-firefox
```
Временные дополнения -> Загрузить временное дополнение

## Использование
После установки регулятор громкости появится рядом с кнопкой действий и чатом в интерфейсе BBB.
